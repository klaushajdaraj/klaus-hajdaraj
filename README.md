# My Portfolio Website

## The stack used on this project

- Next Js
- Typescript
- Blender
- Three Js
- React Three Fiber
- Drei
- React-three-rapier
- MeshLine

## Installation

```bash
npm install
```

### Run the development server

```bash
npm run dev
```

### Build the project

```bash
npm run build
```

### Run the project in production mode

```bash
npm run start
```
